# Quickstart Section

## Badge
Quick Start

## Headline
Running in under a minute

## Description
One package. Three commands. Your brain boots with a website, MCP server, CMS, and chat interface — all in a single process.

## Install Command
$ bun add -g @rizom/brain

## Create Command
$ brain init mybrain --model rover

## Run Command
$ cd mybrain && brain start

## OK Lines
- ✓ Production site + CMS → localhost:8080
- ✓ Preview site + CMS → localhost:4321
- ✓ MCP server → localhost:3333/mcp
